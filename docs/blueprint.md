# **App Name**: SHIELD ERP

## Core Features:

- Dashboard: Display key metrics: Total Clients, Premiums Collected, Active Clients, Inactive Clients, New Business, Outstanding Premiums.
- Client Management: Manage client data: personal details, policy information, payment history.
- Policy Management: Create, update, and track insurance policies.
- Claims Processing: Manage claims: submission, assessment, approval, and payout.
- Financial Reporting: Generate financial reports: revenue, Outstanding premiums, profitability.
- Premium Collection: Track and manage premium collections. Including generating pending payment reminders.

## Style Guidelines:

- Primary color: Deep Blue (#0e5fc9) to convey trust and security, in line with the insurance theme.
- Background color: Light Gray (#F0F4F7), a desaturated version of the primary color, for a clean and professional backdrop.
- Accent color: Teal (#0ec9b4) to highlight key actions and data points, offering a contrasting yet harmonious element.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to create a modern, readable, and professional look.
- Use simple, clear icons to represent dashboard metrics and navigation items.
- A clean, structured layout with clear sections for each ERP function.
- Subtle transitions and animations for data updates on the dashboard.
